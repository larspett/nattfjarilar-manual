# Pilotprojekt nattfjärilar 2026

Fältmanual för pilotprojektet med ljushinksfällor: landskapseffekter (Lund/Uppsala) och latitudgradient (Lund–Abisko).

**Manualen finns här: [larspett.github.io/nattfjarilar-manual](https://larspett.github.io/nattfjarilar-manual/)**

## Om det här repot

- `docs/` — manualens innehåll (Jekyll/GitHub Pages, byggs automatiskt från `main`-branchen)
- `DECISIONS.md` — logg över viktiga beslut i projektet, med motivering
- `CHANGELOG.md` — vad som ändrats i manualen, version för version, plus en backlogg över planerat arbete
